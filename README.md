# Titanic Classification Project

Predicting Titanic passenger survival using multiple machine learning models based on passenger features.

## Project Features

- Data preprocessing and feature scaling
- Logistic Regression, K-Nearest Neighbors, Decision Tree, LDA, QDA, and Naive Bayes models
- Model evaluation with Accuracy and Confusion Matrix
- Saved trained models for later use

## Dataset

- Training data: `data/train_data.csv`  
- Test data: `data/test_data.csv`  

## How to Run

1. Clone the repository:

git clone https://github.com/abarseghian2005-collab/titanic-classification.git

2. Install dependencies:

pip install pandas scikit-learn joblib

3. Run the main script:

python notebooks/titanic_analysis.py

## Example Results

- Logistic Regression → Accuracy ≈ 0.81  
- K-Nearest Neighbors → Accuracy ≈ 0.79  
- Decision Tree → Accuracy ≈ 0.78  
- LDA → Accuracy ≈ 0.80  
- QDA → Accuracy ≈ 0.77
